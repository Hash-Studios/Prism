# Screenshot Placeholders

Add 3–5 app screenshots here named exactly:

- screen1.png
- screen2.png
- screen3.png
- screen4.png  (optional)
- screen5.png  (optional)

Recommended specs:
- Dimensions: 390 × 844 px (iPhone 14 size, 9:19.5 ratio)
- Format: PNG or WebP
- Content: App screenshots — wallpaper browser, detail view, collections, setups, filters

These are referenced in index.html as:
  assets/screenshots/screen1.png … screen5.png
