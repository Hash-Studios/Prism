# Security Policy

## Supported Versions

We support security fixes for the latest public release of Prism.

If you are on an older version, please update first and then verify whether the issue still reproduces.

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Instead, report vulnerabilities privately by emailing:

- hash.studios.inc@gmail.com

Include as much detail as possible:

- A clear description of the issue
- Steps to reproduce
- Affected platform and app version
- Proof of concept or screenshots (if available)
- Potential impact

## Response Process

- We will acknowledge valid reports as soon as possible.
- We will investigate and prioritize fixes based on severity and user impact.
- We will coordinate disclosure timing with the reporter when possible.

Thank you for helping keep Prism and its community safe.
